# xlsx-generator

A Java library for generating XLSX files.

## Requirements

- Java 17+
- Maven 3.8+

## Installation

Add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>ui.github.andryshutka</groupId>
    <artifactId>xlsx-generator</artifactId>
    <version>1.0.0</version>
</dependency>
```

## Usage

```java


XlsxGenerator generator = new XlsxGenerator();
generator.

generate("output.xlsx");
```
